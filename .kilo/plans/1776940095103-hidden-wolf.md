## Technical Specifications

### Splash Screen Design Requirements (from HTML)
- Background: #F8FAF4 with radial gradient overlay
- Grid pattern: 32px spacing, #000000 with 0.03 opacity
- Icon container: 96x96, white, 24px border radius, shadow
- Icon: agriculture symbol, 48px, primary color
- Typography: Manrope 800 for title, Inter 400 for tagline
- Badge: Bottom positioned, pill shape, surface-container background

### Animation Sequence
- 0ms: Icon scale 0.8 → 1.0 (300ms, elasticOut)
- 200ms: Title fade in (400ms)
- 400ms: Tagline fade in (400ms)  
- 600ms: Badge slide up (300ms)
- Total: 1800ms minimum display

### Onboarding Pages
1. "Your Farm, Digitized" - Tracking features illustration
2. "100% Offline" - Offline-first illustration  
3. "Smart Insights" - Analytics illustration

### Form Validation
- Farm name: Required, min 2 chars
- Owner name: Required
- Shed name: Required
- Capacity: Required, positive integer
- Real-time validation with error messages below fields

### Navigation Logic
- Splash: Farm exists → /home/dashboard, else → /onboarding
- Onboarding: Skip/Complete → FarmSetupScreen
- FarmSetup: Valid form → Create farm/shed → /home/dashboard
